組み込み型、組み込み関数など

型: Int, Float, Char, String(= List Char), List a(= <head: a, tail: List a>),
    Eq a, Ord a, Show a

関数:
  #{eq, neq}_{Int, Float, Char}
  #{lt, gt, le, ge}_{Int, Float, Char}

値:
  eqInt = { eq = eq_Int, neq = neq_Int } : Eq Int
  eqFloat = { eq = eq_Float, neq = neq_Float } : Eq Float
  eqChar = { eq = eq_Char, neq = neq_Char } : Eq Char
  eqList (eqA : a) = { eq = _, neq = _ } : Eq (List a)
  ord{Int, Float, Char, List} = _
